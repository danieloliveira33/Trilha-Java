# 🛒 Sisteminha Java - Mercado

Projeto desenvolvido como atividade da disciplina de **Programação Orientada a Objetos**, com o objetivo de praticar os conceitos fundamentais da linguagem Java através da implementação de um sistema de gerenciamento de mercado executado em terminal.

## 📋 Sobre o Projeto

O sistema simula o funcionamento básico de um mercado, permitindo o gerenciamento de produtos, clientes, estoque, vendas e fluxo de caixa.

Todo o projeto foi desenvolvido utilizando apenas recursos fundamentais da linguagem Java, sem frameworks externos, com foco no aprendizado de lógica de programação, estruturas de dados e organização de código.

---

## 🚀 Funcionalidades

### 🔐 Controle de Acesso

* Login de administrador
* Validação de usuário e senha

### 📦 Gestão de Produtos

* Cadastro de novos produtos
* Listagem de produtos cadastrados
* Controle de preços
* Controle de estoque

### 👥 Gestão de Clientes

* Cadastro de clientes
* Listagem de clientes

### 💰 Sistema de Vendas

* Seleção de produtos
* Validação de estoque disponível
* Associação da venda a um cliente
* Geração de recibo
* Atualização automática do estoque

### 🏦 Controle Financeiro

* Registro das vendas realizadas
* Controle de dinheiro em caixa
* Consulta do valor arrecadado

### 📈 Gerenciamento de Estoque

* Reposição de produtos
* Atualização automática das quantidades disponíveis
